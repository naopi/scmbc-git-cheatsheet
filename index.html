add
索引(index)にファイルの内容を追加する
bisect
バグの発生した変更箇所を、二分探索で見つける
branch
ブランチに対する操作を行う
checkout
ブランチまたはパスを作業領域にチェックアウトする
clone
レポジトリを新しいディレクトリに複製する
commit
変更をレポジトリに記録する
dif
コマンドの解説
grep
コマンドの解説
fetch
コマンドの解説
init
リポジトリを作成する
log
コミットログを閲覧する
merge
ローカルブランチのマージを行う
show
コミットの差分・ファイルの表示をする
status
変更が加えられたファイルを表示する
tag
コミットにタグを付ける
mv
ファイル、ディレクトリ、シンボリックリンクを移動または名前を変える
push
公開リポジトリに自分のリポジトリの内容を送信する
pull
ほかのリポジトリの変更点をローカルリポジトリにマージする
diff
違いを示します。
①ワークしているツリーとインデックスやツリーの違い。
②２つのツリーの違い。
③２つのファイルの違い。
grep
パターンにマッチした行を印字します。

インデックスファイルに記録されたblobや指定したツリーオブジェクトのblobからワークツリーに記録された特定のパターンを探します。
特定パターンは１行か改行文字によって区切られたそれ以上の行のリストからなります。
検索対象としての空白文字はすべての行にマッチします

rebase
コミット履歴を変更する インデックスに登録したファイルをコミット対象からはずす。 コミット履歴を変更して、ブランチの分岐元を置き換え コミットの順番の入れ替え コミット履歴の編集
reset
ファイルの変更やコミットをリセットする。
rm
ファイルをバージョン管理からはずす。
